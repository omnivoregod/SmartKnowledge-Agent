# SmartKnowledge-Agent 🤖
**Intelligent Knowledge Base Agent Based on RAG and Custom ReAct Architecture**

## 🚀 Project Overview
This is a personal knowledge base intelligent agent built on the LangChain 1.x LTS architecture. It not only retrieves local PDF documents but also answers complex questions through web search and system tool calls.

The core highlight of the project is that it **does not rely on high‑level abstractions**; instead, it manually implements the ReAct reasoning loop and integrates an **introspection interceptor** at the code level, effectively addressing common “hallucinations” and “task laziness” issues found in lightweight large language models (such as GLM-4-Flash).

## ✨ Key Features
- **Modular Architecture**: Core logic, tools, and data layers are fully decoupled.
- **Custom ReAct Engine**: Manually implements the `Thought -> Action -> Observation -> Final Answer` closed loop.
- **Intelligent Defense Interceptor**: Automatically detects “placeholders” and “perfunctory responses” generated by the model, forcing deeper retrieval.
- **Cross‑Language Support**: Supports English document retrieval and Chinese summary output.

## 🛠️ Tech Stack
- **LLM**: ZhipuAI (GLM-4-Flash)
- **Vector DB**: ChromaDB (langchain-chroma)
- **Framework**: LangChain 1.x LTS, Python 3.13
